### Pipeline

`Pipeline.py` contains the script developed to automate the generation of visualizations, statistical analyses, and the organization of results into a structured folder system.

To run the pipeline, the input datasets must be placed in the `Input` directory using the following file names:

- `data_mix.csv`
- `demand.csv`
- `frequency.csv`

Once executed, the script automatically processes the data, generates the corresponding figures and statistical summaries, and stores the outputs in an organized directory structure according to the analyzed date.
