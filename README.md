# TFM_DiezFernandezMario

This repository contains all the scripts, datasets, and supporting materials used in the Master's Thesis:

**"Low Inertia and Renewable Variability: A Data Science Approach to Quantify Dependencies Between Generation Mix, RoCoF, and OMIE Prices"**

The repository is organized into two main sections:

- **Pipeline/**: Contains the automated analysis workflow described in Chapter 6 of the thesis. This pipeline generates the visualizations, statistics, and reports required for the analysis of a selected day.

- **Analysis/**: Contains the exploratory analyses presented in Chapters 4 and 5, including data preprocessing, statistical studies, visualizations, and the main insights obtained from the datasets.

The project focuses on the analysis of the Spanish power system, studying the relationship between renewable generation, system frequency dynamics, electricity demand, and market behavior, with special attention to the April 2025 Iberian blackout event.
